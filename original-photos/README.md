# Original photos

These are the raw, uncategorized photos and one video from the first
batch of real jobsite pictures used to build the gallery. They are kept
here as the original source files, the versions actually shown on the
live site are separate copies, already sorted into real projects, under
`frontend/public`.

Nothing in the site's code reads from this folder, it is a backup of the
originals, not something the app depends on to run.
